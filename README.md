## 🚀 Overview

Dashy is a self-hostable personal dashboard designed to bring frequently used apps, services, links, status checks, widgets, and visual customization into one central interface. It is intended for users who want a configurable home page for personal infrastructure, home labs, self-hosted services, internal tools, or daily web shortcuts.

The project focuses on flexibility. Dashy includes support for status indicators, widgets, themes, icon packs, a UI editor, authentication, multiple pages and sections, search, shortcuts, alternate views, language switching, and backup and restore options. These capabilities make it suitable for both simple bookmark-style dashboards and more structured service portals.

Dashy’s documentation is organized around running, configuring, managing, developing, and extending the dashboard. The available documentation covers deployment, configuration options, app management, troubleshooting, authentication, theming, widgets, icons, status indicators, and contribution workflows.

## ✨ Key Features

Dashy includes a broad set of dashboard-focused features:

## 🧭 What Dashy Is For

Dashy is useful when you want a single place to access and monitor the tools you use regularly. Instead of keeping links scattered across browser bookmarks, notes, internal documents, or memory, Dashy lets you create a structured dashboard with sections, pages, icons, widgets, and status indicators.

Typical uses include:

- A home dashboard for self-hosted applications
- A home lab landing page
- A personal productivity start page
- A lightweight internal service directory
- A monitoring-oriented dashboard for service availability
- A themed launcher for frequently used links
- A private dashboard protected by authentication
- A multi-page dashboard for separating personal, work, admin, and infrastructure resources

Dashy is not described as a replacement for a full monitoring platform, documentation system, or identity provider. Its purpose is to provide a configurable, self-hostable personal dashboard with practical navigation, visual customization, and status-oriented features.
